# FitnessProject

GAINZ is a web-based fitness tracking application designed to help users plan, record, and analyze their workouts. The platform focuses on personal training management rather than social networking, making it ideal for users who want a clean and structured way to track their progress. The system provides an organized exercise library, customizable training programs, and detailed workout logs that users can review over time. The target audience includes beginner to intermediate gym-goers who want a reliable tool to track sets, reps, weights, and workout history. GAINZ supports secure user accounts with two-factor authentication, a responsive interface for mobile and desktop, and an administrative back-end where admins manage exercises, categories, and built‑in training programs. The scope of the project includes session-based workout logging, analytics, bodyweight tracking, and multi-language support (English and French). Users must be at least 16 years old to create an account.
